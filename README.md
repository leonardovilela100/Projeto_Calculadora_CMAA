# Projeto_Calculadora_CMAA
Calculadora que faz o calculo dos parâmetros de (TCH, Velocidade da Maquina, Tiro Médio, Quantidade de Maquinas)

Criada em Linguagem de programação Java e usando a IDE Netbeans.
